核心功能说明

动态成本价计算与每日复盘工作流

动态成本价计算

系统采用移动加权平均法计算持仓成本，每当执行买入或卖出操作时，系统会自动更新持仓成本，确保成本价准确反映您的实际投入。

最新成本价 = (累计买入金额 - 累计卖出金额) / 当前持仓数量

每日复盘工作流

通过结构化的复盘流程，帮助您系统地记录和分析每日交易，持续提升投资水平。

  1. 选择日期 - 默认当日，可通过日历选择历史日期
  2. 自动填充 - 系统自动拉取大盘数据、持仓信息、当日交易
  3. 板块填写 - 依次填写各复盘板块内容
  4. 保存复盘 - 数据保存至本地，可随时修改
  5. 导出分享 - 支持导出为 Markdown/PDF 格式

使用说明

简单五步，轻松上手

1

添加持仓

点击"添加持仓"按钮，搜索并选择股票，输入买入价格和数量即可添加。

2

交易操作

点击持仓卡片上的"买入"按钮进行加仓，点击"卖出"按钮进行减仓（会显示实现盈亏）。

3

每日复盘

点击"每日复盘"标签页，选择日期，填写大盘指数、持仓情况、操作回顾等板块。

4

查看历史复盘

点击"查看历史"按钮，通过日历选择日期，查看历史复盘内容。

5

刷新行情

点击"刷新价格"按钮获取最新股价，系统会自动更新涨跌幅。

快速开始

几分钟内即可运行

# 克隆项目
git clone https://github.com/TurtleTrace2026/TurtleTrace.git

# 进入项目目录
cd TurtleTrace

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
