核心功能说明

动态成本价计算算法

系统采用移动加权平均法计算持仓成本，每当执行买入或卖出操作时，系统会自动更新持仓成本，确保成本价准确反映您的实际投入。

最新成本价 = (累计买入金额 - 累计卖出金额) / 当前持仓数量

使用说明

简单四步，轻松上手

1

添加持仓

点击"添加持仓"按钮，搜索并选择股票，输入买入价格和数量即可添加。

2

交易操作

点击持仓卡片上的"买入"按钮进行加仓，点击"卖出"按钮进行减仓。

3

刷新行情

点击"刷新价格"按钮获取最新股价，系统会自动更新涨跌幅。

4

查看交易记录

点击持仓卡片底部的"查看交易记录"按钮，查看所有历史交易。

快速开始

几分钟内即可运行

# 克隆项目
git clone https://github.com/TurtleTrace2026/TurtleTrace.git

# 进入项目目录
cd TurtleTrace

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
